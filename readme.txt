=== Automacao de calculos na Calculadora do Android ===

0.Objetivo
 .Realizar calculos basicos na calculadora do Windows,

1.Requisitos:
 .Os drivers devem estar na pasta drivers
 .Java JDK 8
 .Intellij Idea ou Eclipse IDE
 .Dependencias Maven (Ja estao inseridas no arquivo pom.xml pronto para serem baixadas ao importar na IDE
  com as versoes estaveis).

2.Configuracoes:
 .Os metodos para criar o driver Winium estao na classe calculadoraWinCore.

3.Execucao:
 .Abrir uma das classes ***Test.java no Intellij Idea ou no Eclipse
 .No caso para executar numa versao diferente do windows 10, verifique se a calculadora esta no mesmo caminho da calcu-
 ladora do Windows 10.

4.Evidencias:
 .Apos a execucao as imagens de evidencias sao armazenadas na pasta screenshots.

5.Massas:
 .Nao e necessario criar massas por enquanto.
